Introduction

The Hallway Tracker app is a web-based application that allows teachers and administrators to monitor and record student movements in and out of classrooms. It provides real-time tracking, maintains a history of student activities, and offers customizable settings to fit your institution's policies.

Getting Started

Accessing the App

To use the Hallway Tracker app, navigate to the following URL:

https://classbellschedule.com/track

Initial Setup

  1. Open the App: Visit the URL provided above using a modern web browser (Chrome, Firefox, Edge, etc.).
  2. Configure Settings: Click on the Settings button to open the settings modal.
  3. Enter School Details:
    • School Name: Enter the name of your school.
    • App Title: Customize the app title if desired.
    • Teacher Name: Input your name or the supervising teacher's name.
    • Room Number: Specify the room number where the app is being used.
  4. Manage Student List: Upload or enter your student data (explained in detail later).
  5. Save Settings: Click the Close button to save your settings.

Main Features

Student Check-In/Out

  • Student ID Input: Allows students to enter their ID numbers.
  • Reason Selection: Provides a dropdown menu for selecting the reason for leaving (e.g., Restroom, Nurse).
  • GO Button: Checks the student out and starts the timer.
  • I'm Back Button: Checks the student back in and stops the timer.

Currently Out List

  • Displays a real-time list of students who are currently out of the classroom.
  • Shows details such as Name, Grade, Class, Time Out, Duration, and Reason.

Settings

  • School Details: Customize the app with your school's information.
  • Limit Hallway Access: Set limits on how often students can leave the classroom.
  • Student List Management: Upload, edit, and manage the list of students.
  • Auto Export Settings: Configure automatic data export options.

History Tracking

  • History Modal: Access a detailed history of all student movements.
  • Filtering Options: Filter history by date range, class section, and grade.
  • Export and Copy: Export history data to CSV or copy it for use in other applications.

Detailed Usage

Using the Student Check-In/Out System

  1. Student Enters ID:
    • The student inputs their Student ID into the Enter Student ID field.
  2. Select Reason:
    • From the Reason dropdown, select the appropriate reason for leaving.
  3. Click GO:
    • Press the GO button to check out.
    • The system will display a success message and start the timer.
    • The student will appear in the Currently Out list.
  4. Returning to Class:
    • Upon returning, the student re-enters their Student ID.
    • Click the I'm Back button to check back in.
    • The system records the time and removes the student from the Currently Out list.

Note: If a student has exceeded the allowed number of visits (based on settings), the system will display a warning message.

Managing the Student List

Viewing the Student List

  1. Open Settings: Click on the Settings button.
  2. Show Student List: In the Student List card, click Show Student List.
  3. Search and Filter: Use the Filter by Name or ID field to search for students.

Editing the Student List

  1. Enter Edit Mode: Click on the Edit button.
  2. Modify Entries:
    • Click on any cell to edit the information.
    • Delete: Use the Delete button next to a student to remove them.
  3. Save Changes: Click on the Save button to apply changes.
  4. Cancel Editing: Click on Cancel to exit without saving.

Clearing the Student List

While in Edit mode, click the Clear List button to remove all students. Confirm the action when prompted.

Adding Students

Option 1: Copy-Paste Data
  1. Open Settings: Click on the Settings button.
  2. Copy-Paste Option: In the Upload or Copy Student Data card, click Copy-Paste.
  3. Enter Data:
    • Paste your student data into the textarea provided.
    • Use the format:
    Last Name    First Name    Student Number    Grade    Homeroom Code
  4. Process Data: Click Process Data to import the students.
Option 2: Upload File
  1. Open Settings: Click on the Settings button.
  2. Upload Option: In the Upload or Copy Student Data card, click Upload File.
  3. Select File:
    • Click on Choose File and select a CSV, XLS, or XLSX file.
    • Ensure the file contains the correct headers.
  4. Import Data:
    • The app will process the file and import the students.
    • Any duplicates will be reported.

Note: Always ensure your data is correctly formatted to prevent import errors.

Limiting Hallway Access

  1. Open Settings: Click on the Settings button.
  2. Enable Limiting:
    • In the Limit Hallway Access card, check the Do you want to limit GO? checkbox.
  3. Set Limit Period:
    • Choose between Weekly or Date Range.
  4. Configure Limits:
    • Weekly:
      • Max Visits: Enter the maximum number of visits allowed.
      • Week(s): Specify over how many weeks this limit applies.
    • Date Range:
      • Start Date and End Date: Set the date range.
      • Max Visits: Enter the maximum number of visits for the period.
  5. Save Settings: Click Close to apply the changes.

Behavior:

When a student attempts to check out, the app checks their history. If they've exceeded the limit, a warning message is displayed, and they cannot check out.

Exporting Data

Manual Export

  1. Open History: Click on the History button.
  2. Export Data: Click Export to CSV.
  3. Download File:
    • The app will generate a CSV file with the current history data.
    • Save the file when prompted.

Copying Data

  1. Open History: Click on the History button.
  2. Copy Data: Click Copy.
  3. Paste Data:
    • The history data is now copied to your clipboard.
    • You can paste it into Excel or another application.

Auto Export Settings

  1. Open Settings: Click on the Settings button.
  2. Enable Auto Export:
    • In the Auto Export Settings card, check the Auto Export checkbox.
  3. Set Export Period:
    • Choose between Daily or Weekly.
  4. Save Settings: Click Close to apply.

Behavior:

  • Daily Export: The app will automatically export history data at 4:00 PM daily.
  • Weekly Export: The app will export data at 1:00 PM every Friday.
  • The exported file will be saved with a timestamped filename.

Note: Auto export requires the app to be open in a browser to function.

Navigation

  • Home Button: Returns you to the main page (https://classbellschedule.com).
  • Settings Button: Opens the settings modal for configurations.
  • History Button: Opens the history modal to view past records.
  • Filter Controls:
    • Accessed via the Filter button in the history modal.
    • Allows you to filter history data by date, class section, and grade.

Troubleshooting

  • Student Not Found: If a student ID is not recognized, ensure the student is added to the student list.
  • Exceeded Visit Limit: If a student cannot check out due to exceeding limits, review the settings or reset limits as needed.
  • Import Errors:
    • Ensure your data is correctly formatted.
    • Remove any empty rows or invalid entries before importing.
  • Auto Export Not Working:
    • Confirm that the auto export setting is enabled.
    • Ensure the browser remains open for the scheduled export time.

FAQs

Q1: How do I add new students after the initial setup?

A: Go to Settings > Student List > Upload or Copy Student Data and follow the steps to add more students.

Q2: Can I edit a student's information after they've been added?

A: Yes. In the Student List, click Edit, make the necessary changes, and then click Save.

Q3: What happens if I clear the student list?

A: All student data will be removed. You'll need to re-import or re-enter students to use the check-in/out system.

Q4: How is the duration calculated for student outings?

A: The duration is calculated from the time the student clicks GO to the time they click I'm Back.

Q5: Can I customize the reasons for leaving?

A: Currently, the reasons are predefined. Customization may be available in future updates.
